# 🎓 Responsive Student Registration Form

A clean and responsive student registration form built using **HTML, CSS, and JavaScript**.  
This project demonstrates form handling, DOM manipulation, validation, and responsive UI design.

---

## Features

- Fully responsive design (Desktop, Tablet, Mobile)
- Form validation (all fields required)
- Dynamic table row creation using JavaScript
- Prevents page refresh on submit
- Automatic form reset after submission
- Clean and modern UI
- Well-structured and commented code

---

## Technologies Used

- HTML5 (Semantic structure)
- CSS3 (Flexbox + Media Queries)
- JavaScript (DOM Manipulation & Event Handling)

---

## Form Fields

The form includes the following inputs:

- First Name
- Last Name
- Email
- Address
- Pincode
- Gender (Radio Buttons)
- State
- Country

---

## How It Works

1. User fills out the form.
2. On clicking **Submit**:
   - JavaScript validates all fields
   - Prevents page reload using `event.preventDefault()`
   - Captures form data
   - Dynamically adds a new row to the table
3. Form resets automatically after successful submission.

---

## 📱 Responsive Design

- Uses **Flexbox** for layout
- Uses **Media Queries** for mobile responsiveness
- Form stacks vertically on smaller screens
- Table becomes scrollable on mobile devices
